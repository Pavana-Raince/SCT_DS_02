# SCT_DS_02
# Task 02 – Exploratory Data Analysis (EDA)
# Objective
Perform data cleaning and exploratory data analysis (EDA) on a dataset to explore relationships between variables and identify patterns and trends.

# Tools Used
Power BI Desktop

DAX (Data Analysis Expressions)

Netflix Titles Dataset (Kaggle)

# Description
This project focuses on uncovering Netflix's content distribution and acquisition strategy. I performed data cleaning to handle missing values and developed an interactive dashboard to visualize how content type, rating, and country of origin relate to library growth over time.

# Files Included
netflix_analysis.pbix – Power BI project file

netflix_titles.csv – Dataset used

dashboard.png – Final visualization output

# Result
The analysis identifies a significant growth surge between 2016–2019, a strategic shift towards TV-MA rated content, and the dominance of the US and Indian markets in the global catalog.
